【概要】

　日本語プログラミング言語「なでしこ」とは、日本語でプログラムを作ることが出来る
プログラミング言語です。

================================================================================
なでしこは、Borland Delphi 6/7 で記述されています。

dnako.dpr ... なでしこのメインエンジン（ＤＬＬとして実装）
cnako.dpr ... なでしこのコンソール　　（ＥＸＥ）
gnako.dpr ... なでしこの簡易ＧＵＩ版　（ＥＸＥ）
vnako\vnako.dpr ... なでしこのリッチＧＵＩ版（ＥＸＥ）

================================================================================
vnako.dpr には、DelphiのVCLに加え、多くのコンポーネントをインストールして
利用しています。

UIWebBrowser
GIFImage
MAGImage
GldPng が配布終了のため、pngimageにしたけど、GldPngの方が綺麗に画像が出る
TEditor
TEditorEx
TrackBox
XXX_fountain.pas (*)

Indyもインストールしてください。

ActiveXのインポートで、Microsoft Internet Control(MSHTML_TLB) をインストールする必要があります。

など。
