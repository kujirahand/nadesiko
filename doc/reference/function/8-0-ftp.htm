FTPについて

　ＦＴＰではホームページの更新やＷＥＢスペースへのファイル転送を行うことができます。
■ＦＴＰでファイルをアップロード
■必要なファイル
■その他の命令

ＦＴＰでファイルをアップロード

なでしこでＦＴＰへファイルをアップロードするには以下の手順を踏みます。

  • （１）サーバーへ接続
  • （２）ファイルの転送など
  • （３）サーバーとの切断

そこでプログラムは以下のようになります。

！「nakonet.nako」を取り込む
ＦＴＰについて
　　＃接続設定
　　サーバーは「ftp.xxxx.jp」
  　ＩＤは「xxxxx」
    パスワードは「xxxxx」
　　＃実際の処理
    接続
    「{母艦パス}index.html」を「index.html」へアップロード
    切断

必要なファイル

ＦＴＰを使うには、plug-ins\nakonet.dll と lib\nakonet.nako の２つのファイルが必要になります。

その他の命令

ＦＴＰでは、「ダウンロード」「属性変更」「転送モード」「ファイル列挙」「フォルダ列挙」という命令が用意されています。
