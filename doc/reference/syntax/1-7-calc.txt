★計算

=lead
四則演算や比較などを行う演算子とその役割を紹介します。

■四則演算

・A + B　（足し算）

数値同士の足し算を行います。

・A - B  (引き算)

数値同士の引き算を行います。

・A × B （掛け算）

数値同士の掛け算を行います。

・A * B　（掛け算）

数値同士の掛け算を行います。

・A ÷ B （割り算）

数値同士の割り算を行います。

・A / B (割り算)

数値同士の割り算を行います。

・A % B (割り算の余り)

割り算の余りを求めます。

・A ^ B （累乗）

ＡのＢ乗を求めます。

・A & B (文字列の足し算)

文字列同士、足し算を行います。

■比較のための演算子

・A = B (代入/比較)

計算式の中では、AとBが等しいかどうか比較し、
プログラムの構文では、代入を表します。

・A == B (比較：等しい)

計算式の中で、AとBが等しいかどうか比較します。

・A <> B (比較：等しくない)

計算式の中で、AとBが等しくないかどうか比較します。

・A != B, A ≠ B (比較：等しくない)

計算式の中で、AとBが等しくないかどうか比較する。

・A > B (比較：超)

計算式の中で、AがBより大きいか比較します。

・A >= B, A≧B （比較：以上）

計算式の中で、AがB以上かどうか比較します。

・A < B (比較：未満)

計算式の中で、AがBより小さいか比較します。

・A <= B, A≦B （比較：以下）

計算式の中で、AがB以下かどうか比較します。

■ビット操作のための演算子

・A << B （ビット演算）

数値Aを左方向にBビットシフトさせます。

・A >> B （ビット演算）

数値Aを右方向にBビットシフトさせます。

■論理演算のための演算子

・A または B, A || B (理論演算 OR)

比較式Ａが真または比較式Ｂが真ならば真(=1)を返し、
違えば偽(=0)を返します。

・A かつ B, A && B (理論演算 AND)

比較式Ａが真かつ比較式Ｂが真ならば真(=1)を返し、
違えば偽(=0)を返します。
