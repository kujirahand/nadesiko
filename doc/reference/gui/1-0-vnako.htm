標準GUI - vnako.exe

標準GUIの実行環境は、vnako.exeで、若干ファイルサイズが大きくなってしまいますが、ボタン、エディタをはじめ、イメージやカレンダーなどさまざまな部品を利用することができます。

画像処理や描画処理などの命令も vnako.exe に用意されています。

vnako.nako

なでしこのフォルダに、<lib>というフォルダがあります。この<lib>フォルダの中に、vnako.nakoというファイルがあります。

このvnako.nakoの中で、さまざまなGUI部品の宣言が行われています。このvnako.nakoを見ることで、どんな設定項目がどのように定義されているのかを見ることができます。

vnako.nakoの中に多くの『VCL_COMMAND』というのがありますが、このVCL_xxxという命令が、vnako.exeとGUI部品の繋ぎに利用されています。
