リスト

リスト部品は複数の項目を表示しユーザーに選択してもらうことができる部品です。配列変数の値を表示するのにも便利です。
■リストの使い方

リストの使い方

リスト部品は複数の項目から選ぶリストです。以下のように「リストのアイテム」で表示する項目を設定し「リストのテキスト」でどの項目が選ばれているのか取得できます。

寿司リストとはリスト
そのアイテムは「エビ
トロ
アナゴ
イクラ」
これをクリックした時は〜
　　寿司リストのテキストを言う

また、「リストの値」で何番目のアイテムが選択されているのか番号を得ることもできます。

寿司リストとはリスト
そのアイテムは「エビ
トロ
アナゴ
イクラ」
これをクリックした時は〜
　　寿司リストの値を言う

項目が長すぎるときは、『nにリストの横スクロール幅設定』と書いて横スクロールバーを表示させることができます。

寿司リストとはリスト
そのアイテムは「0123456789ABCDEF0123456789ABCDEF
あいうえおかきくえけこ
さしすせそたちつてと」
300にその横スクロール幅設定

長さがどのくらいになるのか分からない場合はクリックしたらヒントを表示するとしてもよいと思います。

寿司リストとはリスト
そのアイテムは「0123456789ABCDEF0123456789ABCDEF
あいうえおかきくえけこ
さしすせそたちつてと」
これをクリックした時は〜
　　寿司リストのヒントは、寿司リストのテキスト。
