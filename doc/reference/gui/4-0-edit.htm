エディタ

　エディタには文字を入力することができます。エディタ系の部品には「エディタ」「メモ」「Tエディタ」「スピンエディタ」があります。
■エディタの使い方
■エディタのイベント

エディタの使い方

エディタを作るのもボタンなどと同じように「ｘｘとはエディタ」と宣言します。

テストエディタとはエディタ。

「エディタ」は一行入力専門のエディタです。「メモ」は複数行入力できる簡易エディタです。「Tエディタ」は本格的なテキストエディタです。「スピンエディタ」は数値入力に便利なエディタです。

それぞれ、「エディタのテキスト」を利用することで値を設定取得することができます。

テストエディタとはエディタ。
テストエディタのテキストは「あいうえお」
テストエディタのテキストを言う。

テキストは省略可能な項目なので、名前だけ書くことでアクセスすることもできます。

テストエディタとはエディタ。
テストエディタは「あいうえお」
テストエディタを言う。

エディタのイベント

エディタには「クリックした時」「ダブルクリックした時」「変更した時」のイベントがあります。

特に「変更した時」は、エディタに何か文字を入力したときに発生するイベントなのでよく使います。

以下はインチ→センチの変換を行うプログラムです。

「インチ」と表示
インチとはスピンエディタ
「センチ」と表示
センチとはスピンエディタ

センチを変更した時は〜インチはセンチ÷2.54
インチを変更した時は〜センチはインチ×2.54
