ボタン

　ボタン部品は押すことができます。普通のボタン「ボタン」と画像を表示できるボタンの「画像ボタン」があります。
■ボタンの使い方
■画像付ボタン

ボタンの使い方

ボタンはGUI部品の中で、最も手軽に使える部品です。以下はボタンの位置とサイズを調節し、クリックするとOKというボタンです。

OKボタンとはボタン。
そのXは10
そのYは10
そのWは300
そのHは300
これをクリックした時は〜
　　「OK」と言う

画像付ボタン

画像付のボタンは『画像ボタン』を使うと作れます。

ワニボタンとは画像ボタン
そのWは400
そのHは300
その画像は「wani.gif」
そのテキストは「ワニを叩け」
これをクリックした時は〜
　　「ガオー！」と言う

画像ボタンはフラット（枠なし）にすることもできます。

ワニボタンとは画像ボタン
そのWは400
そのHは300
その画像は「wani.gif」
そのテキストは「ワニを叩け」
そのフラットはオン
これをクリックした時は〜
　　「ガオー！」と言う
