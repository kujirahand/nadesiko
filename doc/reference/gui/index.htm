　なでしこの標準GUIを利用すると豊富に用意された部品を利用してGUIツールを作ることができます。

★GUI部品の利用法

　GUI部品の多くはなでしこのグループ機能を利用して実装されています。そこで、GUI部品を利用するにはまず『名前とは部品名』としてGUI部品を生成する処理が必要になります。

★母艦について

　なでしこで母艦とはメインフォームのことです。「表示」命令など多くの描画命令で母艦への出力を前提としているものが多くあります。
■母艦とは何？
■母艦の大きさを変える
■デスクトップマスコットにする場合
■閉じるボタンで終わらせたくない場合
■ゲームでキーの入力をチェックしたいとき

★ボタン

　ボタン部品は押すことができます。普通のボタン「ボタン」と画像を表示できるボタンの「画像ボタン」があります。
■ボタンの使い方
■画像付ボタン

★エディタ

　エディタには文字を入力することができます。エディタ系の部品には「エディタ」「メモ」「Tエディタ」「スピンエディタ」があります。
■エディタの使い方
■エディタのイベント

★イメージ

『イメージ』部品は画像を表示したり画像処理をするのに便利な部品です。ぱらぱらアニメを表示するには『アニメ』部品を使います。
■画像を表示する方法
■アニメ部品

★リスト

リスト部品は複数の項目を表示しユーザーに選択してもらうことができる部品です。配列変数の値を表示するのにも便利です。
■リストの使い方

★グリッド

　グリッドは二次元の表（CSVファイルや二次元配列）を表示するのに適した部品です。
