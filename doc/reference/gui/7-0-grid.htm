グリッド

　グリッドは二次元の表（CSVファイルや二次元配列）を表示するのに適した部品です。
■グリッドの使い方

グリッドの使い方

「グリッドのアイテム」を設定することで、グリッドに値を表示できます。

商品グリッドとはグリッド
そのアイテムは「商品名,値段
アイス,200
かき氷,150
バナナ,120
ミカンジュース,250」

編集できるようにするには「編集をオン」に設定します。

商品グリッドとはグリッド
そのアイテムは「商品名,値段
アイス,200
かき氷,150
バナナ,120
ミカンジュース,250」
その編集はオン

選択されている行を取得するには「値」を使います。

商品グリッドとはグリッド
そのアイテムは「商品名,値段
アイス,200
かき氷,150
バナナ,120
ミカンジュース,250」
そのダブルクリックした時は〜
　　商品グリッドの値を言う
