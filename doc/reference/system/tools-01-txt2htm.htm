txt2htmについて

　マニュアルフォルダに、txt2htm.nakoというなでしこのプログラムがあります。これは、手軽にマニュアルを作れるようにと思って作られたものです。普通のテキストファイルと同じように書くことで、ＨＴＭＬのマニュアルを作れます。
■マークアップ
■記述例
■コマンド
　　┣♪ソース
　　┣♪概要
　　┣♪リンク〜(URL)〜(リンクの説明)
　　┣♪ポイント
　　┣♪画像〜(URL)〜(画像の説明)

マークアップ

　普通のテキストと同じように書くと言っても、いくつかの意味のある記号を用います。

  • ★タイトル
  • ■大見出し
  • ●小見出し
  • ♪(コマンド)

記述例

★変数について

♪概要
変数とは数字とか文章とかそういうものを閉まっておける箱のようなものです。

■変数の使い方

変数の使い方は簡単です。

●変数への代入

♪ソース〜？
変数は「変数の内容」
？

●変数の参照

♪ソース〜＄
変数を表示。
＄

...とこのように気楽にソースを書くことができます。

コマンド

コマンドをあらわす♪記号には以下の種類があります。

  • ♪ソース〜(終端記号)
  • ♪概要
  • ♪リンク〜(URL)〜(リンクの説明)
  • ♪ポイント
  • ♪画像〜(URL)〜(画像の説明)

♪ソース

ソースコードを書くときに使います。

♪ソース〜→→
ソース・・・・・・・
・・・・・・・・・・
・・・・・・・・・・
→→

と、ソースコードに使われてない記号を適当に決めて終端文字にすることができます。

♪概要

概要は、「{改行}{改行}」までの部分を切り取って概要とする。

♪概要
概要概要概要概要概要概要概要概要概要
概要概要概要概要概要概要概要概要概要

♪リンク〜(URL)〜(リンクの説明)

リンクを書くときは、『♪リンク〜link.htm〜リンクタイトル』のように書くことができます。

♪ポイント

ポイントも、「{改行}{改行}」までの部分を切り取ってポイントとします。

♪画像〜(URL)〜(画像の説明)

画像を挿入するときは、『♪画像〜(URL)〜(画像の説明)』のように書きます。画像の説明を書くと、IMGタグにALT属性を加え、さらに、画像の右横に解説を添えます。
