★プラグインについて

　なでしこ用のプラグインをplug-insフォルダに入れておくと、なでしこが起動時に自動的にプラグインを読み込み、なでしこに命令や変数・文字列などが追加されます。
■プラグインは何をしている？
■プラグインの作り方

★txt2htmについて

　マニュアルフォルダに、txt2htm.nakoというなでしこのプログラムがあります。これは、手軽にマニュアルを作れるようにと思って作られたものです。普通のテキストファイルと同じように書くことで、ＨＴＭＬのマニュアルを作れます。
■マークアップ
